「iOS」Protocol buffers使用教學231

Protocol Buffer 是 2008 年由 Google 開發的一種跨平台、跨語言的數據交換格式，是一種將結構化資料序列化（變成二進位制）的方法，Google 提供了多種程式語言的實作方式，每一種實作都包含了相對應語言的編譯器以及 Library。

你可以把它想成另一種形式的 JSON ，但資料要比 JSON 格式更小、傳輸更快，所以用於資料傳輸、資料儲存上面會比 XML、JSON 有更大的優勢。
